Number of tests relative to the code size (TST_VOL_IDX)

Description:

The total number of test cases for the product, divided by the number of thousands of SLOC.

Metric is computed from SonarQube's metrics tests and ncloc ).

Test success density (TST_SUCCESS_LAST)

Description:

The percentage of failed tests during the last execution of the test plan.

Computation is as follows:

Test success density = (Unit tests - (Unit test errors + Unit test failures)) / Unit tests * 100

Unit test errors is the number of unit tests that have failed, Unit test failures is the number of unit tests that have failed with an unexpected exception.
